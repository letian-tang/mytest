java8十大新特性
1：接口的默认方法   default标识
2：Lambda 表达式
3：函数式接口 @FunctionalInterface 
        只能有一个抽象方法，但可以overwrite Object基类的方法
   @FunctionalInterface 注解，编译器如果发现你标注了这个注解的接口有多于一个抽象方法的时候会报错的。

5: Lambda 作用域
      在lambda表达式中访问外层作用域和老版本的匿名对象中的方式很相似。你可以直接访问标记了final的外层局部变量，或者实例的字段以及静态变量。
